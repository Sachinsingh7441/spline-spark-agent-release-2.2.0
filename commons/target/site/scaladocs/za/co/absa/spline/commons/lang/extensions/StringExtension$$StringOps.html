  6.  def countUnquoted(charsToFind: Set[Char], quoteChars: Set[Char], escape: Char = '\\'): Map[Char, Int]

    Counts the occurrences of the chars to find.

    Counts the occurrences of the chars to find. The occurrence is not considered if the character is part of a sequence within a pair of quote characters specified by quoteChars param. Escape character in front of a quote character will cancel its "quoting" function. Escape character in front of a searched-for character will not result in positive identification of a find Double escape character is considered to be escape character itself, without its special meaning The escape character can be part of the charsToFind set or quoteChars set and the function will work as expected (e.g. double escape character being recognized as a searched-for character or quote character), but it cannot be both - that will fire an exception.

    charsToFind

    set of characters to look for

    quoteChars

    set of characters that are considered as quotes, everything within two (same) quote characters is not considered

    escape

    the special character to escape the expected behavior within string

    returns

    map where charsToFind are the keys and values are the respective number of occurrences
