  6.  object SparkLineageInitializer

    Spline agent initializer - enables lineage tracking for the given Spark Session.

    Spline agent initializer - enables lineage tracking for the given Spark Session. This is an alternative method to so called "codeless initialization" (via using spark.sql.queryExecutionListeners config parameter). Use either of those, not both.

    The programmatic approach provides a higher level of customization in comparison to the declarative (codeless) one, but on the other hand for the majority of practical purposes the codeless initialization is sufficient and provides a lower coupling between the Spline agent and the target application by not requiring any dependency on Spline agent library from the client source code. Thus we recommend to prefer a codeless init, rather than calling this method directly unless you have specific customization requirements.

    (See the Spline agent doc for details)
